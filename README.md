# 🏢 기업 인사관리 시스템 (HR System)

본 프로젝트는 Java와 Oracle DB를 활용하여 기업 내 인사 업무를 통합 관리하는 **JDBC 기반의 콘솔형 인사관리 프로그램**입니다[cite: 1]. 기업의 인사 흐름을 데이터베이스 중심으로 설계하고 실무 시스템 개발 역량을 키우는 것을 목표로 하였습니다

---

## 🛠️ 개발 환경
* **Language**: Java
* **Database**: Oracle (DBMS)
* **IDE**: Eclipse
* **Management**: Git/GitHub

---

## 👨‍👩‍👧‍👦 팀원 소개
* **2조**: 진민석, 김동운, 정윤호

---

## 💡 개발 목적
* Java와 데이터베이스 연동 기술(JDBC)의 실전 학습 및 CRUD 구현
* 기업 인사 업무 프로세스(인사발령, 근태, 급여 등)의 시스템화 이해

---

## 📋 주요 기능

### 👤 사용자(사원) 기능
* **내 정보 관리**: 정보 조회 및 개인 정보 수정
* **근태 관리**: 출/퇴근 기록, 월별 근태 현황 및 근무시간 조회
* **조직도 조회**: 부서별 조직 현황 및 연락처 확인
* **게시판**: 사내 공지 확인, 게시글 작성 및 투표 참여

### 🛡️ 관리자 기능
* **사원 관리**: 사원 승인/거절, 정보 수정 및 재직 상태 관리
* **부서/직급 관리**: 부서/직급 신설, 폐기, 수정 및 직급별 기본급 설정
* **인사발령 관리**: 인사발령 이력 기록 및 유형별/기간별 조회
* **급여 관리**: 수당(야근/휴일) 등록, 급여 지급 여부 관리 및 총 급여 조회
* **근태 관리**: 특정 사원 근태 조회 및 데이터 수정
* **통계**: 인사/근태/급여 관련 기간별/부서별/직급별 통계
* **로그 기록**: 로그인 로그 및 사용자 행동 로그 조회(보안)
